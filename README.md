# АРТИДА — artida.by

Корпоративный сайт ООО «АРТИДА» — белорусского производителя электроники собственной разработки и контрактного производства.

**Статус:** v2 в разработке (v1 архивирована)

---

## О компании

АРТИДА — производственная компания из Минска, которая:

- **Выполняет контрактное производство** электроники под заказ — это главный продукт для продвижения
- **Разрабатывает и производит** собственную электронику (текущая ниша — СКУД: бесконтактные кнопки выхода, считыватели карт, контроллеры)

Основные продукты: МАГИЯ-3 (бесконтактная кнопка с ИК-сенсором), КТМ-УД (считыватель карт), КЕ-01 (контроллер).

---

## Назначение сайта

Витрина производственных компетенций + воронка в прямой контакт. Не интернет-магазин.

**Целевая аудитория:** системные интеграторы СКУД, заказчики контрактного производства (Беларусь, Россия, Казахстан).

**Главное действие посетителя:** позвонить или написать на почту.
